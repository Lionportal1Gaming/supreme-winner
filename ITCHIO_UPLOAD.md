# Itch.io Upload Guide - v0.1.1

## Step 1: Create Itch.io Account
- Go to https://itch.io
- Sign up or log in

## Step 2: Create New Project

1. Go to https://itch.io/game/new
2. Fill in the details:

### Basic Information
- **Title:** Project Aeterna
- **Project URL:** `your-username.itch.io/project-aeterna`
- **Short description:** "A text-based dynastic strategy game where you build and maintain a legacy across generations"
- **Classification:** Game
- **Kind of project:** Downloadable

### Pricing
- **Pricing:** Free (or set a price/donations)
