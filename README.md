# Project Aeterna: BloodLines (Alpha)

Welcome to the alpha release of **Project Aeterna**, a text-based grand strategy game of dynastic legacy.

## Overview
Guide your dynasty through the ages, from the founding of Rome to the distant future. Manage your family, expand your influence, and ensure your bloodline survives the trials of history.

## Installation

### Method 1: Standalone Executable (Recommended)
1.  Download the latest release.
2.  Run the `BloodLines` executable.
3.  No Python installation required!

### Method 2: Running from Source
1.  Ensure you have Python 3.8+ installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the game:
    ```bash
    python main.py
    ```

## Controls
-   **[ENTER]**: Advance Time (Next Month)
-   **[M]**: Move Character
-   **[Q]**: Quit Game
-   **Event Selection**: Type the number of your chosen option when an event occurs.
